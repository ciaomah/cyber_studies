# CIAO_CYBER

Este projeto foi feito como parte de um desafio de estudos. O objetivo inicial era criar algo simples para exercicio sobre o conteudo, porém aproveitei a oportunidade para explorar um design mais elaborado e dar personalidade na página.

---
Durante o desenvolvimento utilizei ajuda da IA para acelerar e criar o corpo do projeto deixando-o mais completo, assim explorando ideias de design.

Participei ativamente do processo realizando adaptacoes necessárias, ajuste na estrutura, modificacoes no layout, testes e correcoes de acordo com o meu gosto pessoal.

Também aproveitei este projeto para relembrar conhecimentos de HTML e CSS que havia adquirido anteriormente em um curso na qual não cheguei a concluir, com a memória ainda fresca utilizei o desafio como uma oportunidade prática de aprendizado.
